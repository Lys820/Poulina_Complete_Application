Objectif de l'assistant

L'assistant IA Poulina est un service de questions-réponses specialise dans le domaine avicole au sein du groupe Poulina. Il exploite une base de connaissances composee des historiques d'analyses de laboratoire et des informations sur les laboratoires partenaires, ainsi qu'un modele de recommandation de souches (races de volailles) base sur vos parametres de production.

Capacites disponibles

Questions generales

Posez toute question relative aux analyses, maladies avicoles, normes sanitaires ou fonctionnement des laboratoires.

Recommandation de souche

Renseignez vos parametres de production (type, gouvernorat, budget, biosecurite) pour obtenir une recommandation ML personnalisee.

Filtre geographique

Utilisez le champ "Filtrer par ville" pour restreindre les resultats de laboratoires a une region precise.

Alertes sanitaires

Interrogez l'assistant sur les alertes ou incidents sanitaires recents enregistres dans la base de donnees.

Exemples de questions

Analyses
  • "Quels sont les resultats d'analyse pour le centre de Bizerte ?"
  • "Quelle est la norme de taux de mortalite acceptable pour des poulets de chair ?"
  • "Y a-t-il des alertes sanitaires actives dans la region de Sfax ?"
Laboratoires
  • "Quel laboratoire est disponible rapidement a Tunis ?"
  • "Quels laboratoires traitent les analyses bacteriologiques ?"
  • "Coordonnees du laboratoire de Sousse ?"
Recommandation de souche
  • "Quelle souche recommandes-tu pour un elevage de 10 000 poulets a Nabeul avec un budget de 80 000 DT ?"
  • "Souche optimale pour une production d'oeufs avec un score de biosecurite de 7 ?"

Bonnes pratiques

+ Formulez des questions precises en mentionnant la region, le type de production ou le contexte pertinent.
+ Pour une recommandation de souche, fournissez autant de parametres que possible (budget, biosecurite, taux de mortalite acceptable).
! L'assistant est limite au domaine avicole et a la base de donnees Poulina. Les questions hors domaine recevront une reponse negative.
! La session est reinitialise a chaque rechargement de page. Utilisez le bouton "Reinitialiser" pour demarrer une nouvelle conversation.
